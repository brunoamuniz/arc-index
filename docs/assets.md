# Assets Setup Instructions

## Favicon

The project uses `favicon.svg` as the primary favicon. For maximum browser compatibility, you can generate a `.ico` file:

1. Visit [RealFaviconGenerator](https://realfavicongenerator.net/)
2. Upload `favicon.svg`
3. Generate and download the favicon package
4. Place `favicon.ico` in the `/public` directory

Alternatively, you can use online tools or ImageMagick:
```bash
convert favicon.svg -resize 32x32 favicon.ico
```

## Open Graph Image

The Open Graph image is automatically generated by Next.js using `app/opengraph-image.tsx`. 

If you want to use a static image instead:
1. Create a 1200x630px PNG image
2. Save it as `public/og-image.png`
3. Update `app/layout.tsx` to reference `/og-image.png` instead of `/opengraph-image`

## Apple Touch Icon

The `apple-icon.png` (180x180px) is used for iOS home screen icons. Make sure it's properly sized and optimized.
